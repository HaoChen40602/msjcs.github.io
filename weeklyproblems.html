7. Create a program that outputs a random binary maze (to out7.txt), given an input height and width. The input file “in7.txt” will contain the width followed by the height separated by a space. You are to output the maze in a format like this (for example, input “6 6”):
OXOOOX
OXOXOX
OOOXXX
OXOOOX
OXOXOX
XXXXOO
Your maze must reduce to a connected tree (no simple cycles). “X” represents a wall, and “O” represents a path. Your maze must not include a 4 by 4 square or any simple cycle of X’s (and no 4 by 4 squares of O’s, since that would be a simple cycle). The start of the maze is the top-left corner and the end is the bottom-right. (simple cycle is here defined as a cycle in a graph that does not go through any point twice)

8. [This is a bonus!]Create a program that takes a maze like one generated from problem 8 (in8.txt) and outputs (to out8.txt) a series of U,D,L and R such that following the directions from the start of the maze will allow one to finish the maze without crashing into a wall.
